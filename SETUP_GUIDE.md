# Weather Dashboard - Setup & Usage Guide\n\n## 🚀 Quick Start\n\n### 1. Get API Key\n1. Visit [OpenWeatherMap](https://openweathermap.org/api)\n2. Create a free account\n3. Go to API Keys section\n4. Copy your API key\n\n### 2. Setup Project\n```bash\n# Install dependencies\nnpm install\n\n# Create .env file\ncp .env.example .env\n\n# Edit .env and add your API key\nVITE_OPENWEATHERMAP_KEY=your_key_here\n```\n\n### 3. Run Development Server\n```bash\nnpm run dev\n```\n\nOpen http://localhost:5173 in your browser.\n\n---\n\n## 📋 Features Implemented\n\n### ✅ Core Features\n- [x] Real-time weather data fetching from OpenWeatherMap API\n- [x] Current weather display (temperature, conditions, \"feels like\")\n- [x] 5-day weather forecast\n- [x] Detailed weather metrics (humidity, wind speed, visibility, pressure)\n- [x] Search functionality by city name\n- [x] Error handling for invalid cities\n- [x] Loading states and spinners\n\n### ✅ Advanced Features\n- [x] Recent search history (last 5 cities)\n- [x] Unit toggle (Celsius/Fahrenheit, m/s/mph)\n- [x] Responsive design (mobile, tablet, desktop)\n- [x] Weather icons from Lucide React\n- [x] Smooth animations and transitions\n- [x] Professional gradient UI\n- [x] TypeScript for type safety\n- [x] Proper data validation and processing\n\n---\n\n## 🛠️ Tech Stack\n\n- **React 18** - UI framework\n- **TypeScript** - Type safety and better development experience\n- **Vite** - Fast build tool and dev server\n- **Lucide React** - Beautiful weather icons\n- **CSS3** - Modern styling with flexbox and grid\n\n---\n\n## 📱 Responsive Breakpoints\n\n- **Desktop**: 1200px+ (full features)\n- **Tablet**: 768px-1199px (optimized layout)\n- **Mobile**: 480px-767px (stacked layout)\n- **Small Mobile**: <480px (minimal layout)\n\n---\n\n## 🎨 Design Features\n\n### Color Scheme\n- Primary Gradient: `#667eea` to `#764ba2` (purple to indigo)\n- Text: `#333` (dark gray) and `#666` (medium gray)\n- Backgrounds: White and light gradients\n- Accents: Blue and purple tones\n\n### Typography\n- Heading: 2.5rem (bold)\n- Title: 1.5rem (semibold)\n- Body: 1rem (regular)\n- Small: 0.9rem (light)\n\n---\n\n## 📡 API Endpoints Used\n\n### Current Weather\n```\nGET /weather?q={city}&appid={key}&units={metric|imperial}\n```\n\nReturns:\n- Temperature, humidity, pressure\n- Wind speed and direction\n- Visibility and weather conditions\n- Sunrise/sunset times\n\n### 5-Day Forecast\n```\nGET /forecast?q={city}&appid={key}&units={metric|imperial}\n```\n\nReturns:\n- 40 entries (5 days × 8 per day)\n- 3-hour interval forecasts\n- Temperature, humidity, precipitation\n\n---\n\n## 🔧 Configuration Options\n\n### Change Default City\nEdit `src/components/WeatherDashboard.tsx`:\n```typescript\nconst [searchCity, setSearchCity] = useState('Tokyo');\n```\n\n### Change Default Units\n```typescript\nconst [unit, setUnit] = useState<'metric' | 'imperial'>('imperial');\n```\n\n### Customize Colors\nEdit `src/components/WeatherDashboard.css`:\n```css\n.weather-dashboard {\n  background: linear-gradient(135deg, #your-color-1 0%, #your-color-2 100%);\n}\n```\n\n---\n\n## 📊 Performance Metrics\n\n- Check spelling\n- Try with country code: \"Paris, FR\"\n- Check OpenWeatherMap API documentation\n\n### API Key Issues\n- Verify key in .env file\n- Check key has sufficient credits\n- Ensure key is
